== 상품

=== 상품 등록
상품 등록에 필요한 데이터 `JSON` 객체를 HTTP 메소드 `POST` 요청으로 상품을 등록합니다.

- 인증 토큰을 `Header` 에 담습니다. ( Authorization: Bearer <인증 토큰> )
